# ContainerRestartPolicy

Specifies the policy for restarting containers when they exit or fail.

**Properties**

| Name       | Type   | Required | Description  |
| :--------- | :----- | :------- | :----------- |
| always     | string | ✅       | "always"     |
| on_failure | string | ✅       | "on_failure" |
| never      | string | ✅       | "never"      |
