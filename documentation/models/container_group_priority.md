# ContainerGroupPriority

Specifies the priority level for container group execution, which determines resource allocation and scheduling precedence.

**Properties**

| Name   | Type   | Required | Description |
| :----- | :----- | :------- | :---------- |
| high   | string | ✅       | "high"      |
| medium | string | ✅       | "medium"    |
| low    | string | ✅       | "low"       |
| batch  | string | ✅       | "batch"     |
